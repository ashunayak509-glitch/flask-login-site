# Flask Login & YouTube Video Site

A simple Flask web application that allows users to log in and view embedded YouTube videos from a single channel. This project demonstrates Python web development with Flask, HTML/CSS, session management, and embedding media.

---

## Features

- Login system with username and password  
- Logout functionality  
- Home page displaying YouTube video thumbnails and embedded videos  
- Responsive, colorful, and modern UI with animations  
- Clean external CSS shared between login and home pages  

---

## Tech Stack

- Python 3  
- Flask  
- HTML / CSS (External stylesheet for styling)  
- Jinja2 templates

## How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/yourusername/flask-login-site.git
cd flask-login-site

2. Create a virtual environment (optional but recommended):
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows


3. Install dependencies:
pip install -r requirements.txt

4. Run the app:
python app.py

Open your browser:
http://127.0.0.1:5000
